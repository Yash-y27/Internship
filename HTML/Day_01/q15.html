Introduction

HTML5 is the standard markup language used to create web pages. It provides semantic elements that make web pages easier to understand for both developers and search engines.

Why Use Semantic HTML?

Semantic elements improve accessibility, readability, and SEO by describing the meaning of the content.

"Semantic HTML gives meaning to your content rather than simply defining its appearance."

Example Code

Here is a simple HTML heading:


                    <h1>Welcome to HTML5</h1>

Another example showing a paragraph:


                    <p>This is my first web page.</p>

Conclusion

Learning semantic HTML is the first step toward becoming a skilled web developer. It helps create well-structured, accessible, and maintainable websites.
