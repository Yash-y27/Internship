<html>
The root element that contains the entire HTML document.
<head>
Contains metadata such as the title, styles, and links.
<title>
Sets the title displayed in the browser tab.
<body>
Contains all the visible content of the webpage.
<h1>
Defines the main heading of a webpage.
<p>
Represents a paragraph of text.
<a>
Creates a hyperlink to another page or resource.
<img>
Displays an image on the webpage.
<ul>
Creates an unordered (bulleted) list.
<ol>
Creates an ordered (numbered) list.
<li>
Represents a list item inside an ordered or unordered list.
<table>
Creates a table to display data in rows and columns.
<form>
Creates a form for collecting user input.
<input>
Creates an input field such as a text box, checkbox, or password field.
<button>
Creates a clickable button for submitting forms or performing actions.
